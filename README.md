# Retail Sales Data Analysis
## Description
This project uses actual sales data from retail business (which could be furniture or office equipment) to give hands-on experience with EDA and business analytics.
## Practice Skills
- Data Processing using Pandas: splitting column, converting order date to datetime format, merging multiple dataframes for a master table.
- Data Visualization & Analysis: calculate KPIs for the summary, visualize sales trend, top products, order status, customer age, payment method preference, and shipping method.
